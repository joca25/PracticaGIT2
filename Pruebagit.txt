sistema de control escolar
_______________________________
este sistema permite gestionar las actividades de control escolar,tales como
captura de alumnos, calificacion y boleta.

Requisitos funcionales
________________________________________________
1.-El sistema debe permitir el registro de alumnos
2.-El sistema debe mostrar reporte de alunmos dados de baja
3.-El sistema debe permitir la captura de calificaciones

Requisitos no funcionales
________________________________
1.-el sistema debe ser facil de usar
2.-El sistema debe manejar datos de manera segura
3.- El sistema debe de estar disponible todo el tiempo
